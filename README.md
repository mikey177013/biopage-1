# PHOENIX PAGE

A cyberpunk-style interactive terminal portfolio with Matrix-style animations and hacker aesthetic.

# Features

· Interactive Terminal Interface: Simulates a hacker terminal with command-line aesthetics
· Matrix Background: Animated falling binary code (0s and 1s) in green
· Scan Line Effect: Moving horizontal scan line for authentic hacker terminal feel
· Glitch Animations: Text glitch effects for cyberpunk style
· Progressive Loading: Custom loading screen with progress bar
· Audio Integration: Background hacker-themed music (auto-plays on access)
· Responsive Design: Single-column layout that works on all devices
· Interactive Elements: Clickable skills and info items with visual feedback

# Technologies Used

· HTML5
· CSS3 (with animations and transitions)
· Vanilla JavaScript
· Canvas API for Matrix background

# Sections

1. Access Screen: Initial login screen with "ENTER SYSTEM" button
2. About Me: Personal information (Nickname: PHOENIX, Specialization: DEVELOPER, Experience: 1.5 YEARS)
3. My Knowledge: Skills grid (C++, HTML, PYTHON, JS, MONGODB, VERCEL, RENDER, NODEJS)
4. Contact: Contact information (Email and Telegram)

# Installation

1. Clone the repository or download the HTML file
2. Open index.html in any modern web browser
3. No additional dependencies required

# How to Use

1. Open the page in your browser
2. Click "ENTER SYSTEM" on the access screen
3. Watch the loading animation and progress bar
4. Explore the interactive terminal interface
5. Click on any skill or info item to see interactive feedback

# Customization

Modify the CSS in the <style> section to change:

· Animations: Adjust timing in @keyframes rules
· Layout: Modify grid configurations and spacing
